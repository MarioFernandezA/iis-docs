# ClearManifestContext
Clears the manifest context created using an earlier call to [ApplyManifestContext](../../\express-api-reference/applymanifestcontext.md).  
  
## Syntax  
  
```cpp  
    HRESULT  
    ClearManifestContext();  
```  
  
## Return Value  
 `HRESULT`
