# ClearManifestContext
Clears the manifest context created using an earlier call to [ApplyManifestContext](../../extensions\express-api-ref/applymanifestcontext.md).  
  
## Syntax  
  
```cpp  
    HRESULT  
    ClearManifestContext();  
```  
  
## Return Value  
 `HRESULT`
